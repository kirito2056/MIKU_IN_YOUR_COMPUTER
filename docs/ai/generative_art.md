# 생성형 아트 (Generative Art)

## 1. Stable Diffusion 통합
-   **모델**: SDXL Turbo or SD 1.5 (속도/VRAM 고려).
-   **목적**:
    -   **그림 그리기**: "나 그려줘", "고양이 그려줘" 요청 처리.
    -   **꿈 일기**: 수면 모드 중 생성된 '꿈' 시각화.
-   **UI**: 생성된 이미지는 오버레이 윈도우 한켠에 '가상 캔버스'를 띄워 보여줌.

## 2. Workflow
1.  사용자 요청 ("그림 그려줘").
2.  LLM이 프롬프트 최적화 (English prompt generation).
3.  SD 파이프라인 호출.
4.  이미지 생성 및 클라이언트로 전송.
