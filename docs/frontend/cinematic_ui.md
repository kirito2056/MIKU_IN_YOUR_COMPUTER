# Cinematic UI Interactions

## 1. Physical Interactions (물리 상호작용)
-   **Window Parkour**: 윈도우 창 제목 표시줄을 밟고 뛰어다님.
-   **Icon Physics**: 발로 차면 아이콘이 미끄러짐.
-   **Cursor Hunting**: 마우스 커서를 손으로 잡으려 함.
-   **Skirt Kick**: 치마를 들추면 마우스 커서를 화면 밖으로 걷어참.
-   **Headpat**: 머리를 쓰다듬으면 눈을 감고 좋아함.

## 2. Magic & Abilities (능력)
-   **Telekinesis (염력)**: 손짓으로 윈도우 창을 정리하거나 닫음.
-   **Materialization (물질화)**: 음식 사진을 주면 먹고, 꽃 사진은 화병에 꽂음.
-   **Glow Stick**: 어두운 곳(불 끄면)에서 야광봉을 흔듦.

## 3. Visual Effects (연출)
-   **Screen Wipe**: 주인이 울면 화면 안쪽에서 유리를 닦아줌 (Glass Shader).
-   **Screen Knock**: 주인이 멍때리면 화면을 똑똑 두드림.
-   **Seasonal Decor**: 크리스마스엔 눈, 봄엔 벚꽃 (Overlay Particle).
-   **Confetti**: 깃허브 푸시 성공 시 축하 폭죽.
-   **Slow Motion**: 시스템 렉 걸리면 행동도 느려짐.
-   **Glitch**: 오류 발생 시 텍스처 깨짐 효과.

## 4. Special Modes
-   **SAO Menu**: 허공에 홀로그램 메뉴 전개 (환복 등).
-   **Chair Dance**: 음악 감상 시 의자 소환 & 세라핀 댄스.
-   **Battle Mode**: 바이러스 감지 시 대파(Leek) 소환 후 타격.
-   **Graffiti**: 심심할 때 화면에 크레파스로 낙서.
-   **Watch Party**: 유튜브 감상 시 옆에 앉아서 같이 봄.
-   **Mirror Mode**: 웹캠을 통해 사용자의 표정을 따라 함.
