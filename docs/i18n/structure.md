# 국제화 전략 (Internationalization)

## 1. 지원 언어
-   **한국어 (ko-KR)**: 기본(Default). 미쿠의 성격(드립, 밈)이 가장 잘 반영됨.
-   **일본어 (ja-JP)**: "오리지널 미쿠" 느낌. 경어체/반말 모드 지원.
-   **영어 (en-US)**: 글로벌 대응.

## 2. 파일 구조
`i18next` 라이브러리 표준을 따릅니다.

```
/frontend/src/locales/
  ├── ko/
  │   ├── common.json  (UI 버튼, 메뉴)
  │   ├── dialog.json  (시스템 대화, 에러 메시지)
  │   └── miku.json    (미쿠의 고정 대사 - 인사, 알림)
  ├── ja/ ...
  └── en/ ...
```

## 3. 동적 언어 감지
-   **OS 언어 동기화**: 최초 실행 시 시스템 언어 감지.
-   **대화 언어 자동 전환**: 사용자가 일본어로 말을 걸면 LLM이 자동으로 일본어로 답변하며, UI 힌트도 일본어로 변경 가능(옵션).

## 4. TTS 언어 처리
-   `ko`: 한국어 파인튜닝 모델.
-   `ja`: 일본어 파인튜닝 모델 (오리지널 성우 톤).
-   `en`: 영어 억양 학습 모델 (약간의 일본어 억양이 섞인 귀여운 영어).
